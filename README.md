# 🤖 J.A.R.V.I.S. - Voice Automation Phase 1

Assistente virtual desenvolvido em Python focado em automação de desktop e controle residencial via comandos de voz.

## 🌟 Destaques do Projeto
* [cite_start]**Domótica (IoT):** Controle de dispositivos inteligentes (lâmpadas) via protocolo local.
* [cite_start]**Controle de voz:** Reconhecimento e síntese de voz em tempo real para interação[cite: 1, 5, 15].
* [cite_start]**Automação de Desktop:** Abertura de sites e aplicações predefinidas através de comandos verbais[cite: 5, 18, 20].

## 🛠️ Tecnologias
* Python 3.11
* Biblioteca TinyTuya (Integração Smart Home)
* SpeechRecognition & gTTS (Interface de Voz)
* AppOpener (Automação de Software)
